### LOGO设计
<img width="2048" height="2048" alt="logo" src="https://github.com/user-attachments/assets/69a0b499-c4a9-42a9-ab93-bafe4d29c6b1" />
- 核心设计理念
这款1:1比例的logo，核心围绕「化学专业为根，软件工程为工具，用代码赋能化学学习」的专属定位设计：
以化学领域通用的苯环六元结构为基底，锚定你的化学专业身份；将编程通用的核心语法符号融入苯环顶点，用化学键线条串联，实现双学科内核的自然融合，而非生硬拼接。
主色选用低饱和渐变紫色，既贴合化学经典显色的专属记忆点，又兼具编程领域的科技感，风格温柔沉稳不浮夸。
全程采用统一圆角线性设计，贴合两个学科的严谨特质，1:1对称居中的结构，可直接适配头像、项目封面等全场景，缩放无变形。
- 简洁版设计过程
需求锚定：明确核心诉求——化学专业大学生，跨界学习软件工程以辅助本专业，需1:1比例、紫色为主、简约线性风格，适配GitHub学习主页，拒绝浮夸设计。
元素融合设计：筛选化学端的苯环六元结构、编程端的通用代码符号为核心元素，将代码符号替换苯环顶点，用化学键线条串联，实现双学科元素的无缝结合。
规范与比例把控：锁定1:1正方形画布，以中心为圆心做对称设计；统一圆角线性风格，通过主辅线宽分层区分结构与细节，保证小尺寸下依然清晰，视觉干净柔和。
配色定版与适配：选定低饱和紫渐变作为唯一主色，测试深浅背景下的显示效果，确保全场景清晰可用，同时贴合温柔不刺眼的风格要求。
优化落地：调整元素平衡，删除所有冗余装饰，最终输出1:1矢量文件，可适配头像、笔记、项目封面等全学习场景使用。
